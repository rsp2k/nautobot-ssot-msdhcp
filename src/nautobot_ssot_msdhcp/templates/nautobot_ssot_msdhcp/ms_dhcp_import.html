Import Microsoft DHCP

Bring a Microsoft (Active Directory) DHCP server's scopes, reservations, options, and leases into Nautobot. Run the export on Windows, then upload it here — the import runs as the Microsoft DHCP → Nautobot SSoT job, with a full log and a reviewable diff.

1 Download the script

Export-MSDHCP.ps1 uses the built-in DhcpServer PowerShell module (present on a DHCP server, or via RSAT) to dump the config to JSON. It is read-only — it changes nothing on the server.

Download {{ script_name }}
2 Run it on the server

In an elevated PowerShell prompt, as a user with DHCP read access:

.\Export-MSDHCP.ps1 -ComputerName dhcp01 -OutFile C:\temp\dhcp01.json
-ComputerName
server to query (defaults to the local machine)
-OutFile
where to write the JSON
-IncludeLeases $false
skip dynamic leases if they're large

Export each server separately; for a failover pair, run it on both partners.
